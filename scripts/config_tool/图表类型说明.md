## 📊 对比类图表

### 1. 横向条形图 (bar_horizontal)

**用途**：分类数据对比，适合类别较多的场景

**配置**：
```json
"CHART:sales_by_person": {
  "data_source": "销售员业绩",
  "chart_type": "bar_horizontal",
  "x_field": "总销售额",
  "y_field": "销售员",
  "title": "销售员业绩表现分析"
}
```

**字段要求**：
- `x_field`: 数值字段（如：销售额）
- `y_field`: 分类字段（如：销售员）

**适用场景**：
- ✅ 销售员业绩排名
- ✅ 城市销售对比
- ✅ 产品销量对比
- ✅ 类别超过 5 个的对比

**示例**：
```
数据源：销售员业绩
X 轴：总销售额
Y 轴：销售员
结果：9 个销售员的横向条形图，自动添加数据标签
```

### 2. 纵向柱状图 (bar_vertical)

**用途**：分类数据对比，适合类别较少的场景

**配置**：
```json
"CHART:monthly_sales": {
  "data_source": "月度趋势",
  "chart_type": "bar_vertical",
  "x_field": "年月",
  "y_field": "总销售额",
  "title": "月度销售对比"
}
```

**字段要求**：
- `x_field`: 分类字段（如：月份）
- `y_field`: 数值字段（如：销售额）

**适用场景**：
- ✅ 月度销售对比
- ✅ 季度业绩对比
- ✅ 类别少于 10 个的对比

**示例**：
```
数据源：月度趋势
X 轴：年月
Y 轴：总销售额
结果：6 个月的纵向柱状图，自动添加数据标签
```

---

### 3. 多列柱状图 (column_clustered)

**用途**：多系列数据对比

**配置**：
```json
"CHART:customer_comparison": {
  "data_source": "客户类型",
  "chart_type": "column_clustered",
  "category_field": "客户属性",
  "series": ["总销售额", "订单数", "客单价"],
  "title": "新老客销售贡献对比"
}
```

**字段要求**：
- `category_field`: 分类字段
- `series`: 数值字段数组（2-4 个）

**适用场景**：
- ✅ 新老客对比（多指标）
- ✅ 多产品线对比
- ✅ 多地区对比

**示例**：
```
数据源：客户类型
分类：客户属性
系列：总销售额、订单数、客单价
结果：新老客的多指标对比柱状图
```

---

### 4. 气泡图 (bubble)

**用途**：三维数据对比

**配置**：
```json
"CHART:product_analysis": {
  "data_source": "产品分析",
  "chart_type": "bubble",
  "x_field": "销售额",
  "y_field": "利润率",
  "size_field": "订单数",
  "title": "产品销售额 - 利润率 - 订单数分析"
}
```

**字段要求**：
- `x_field`: X 轴数值字段
- `y_field`: Y 轴数值字段
- `size_field`: 气泡大小字段

**适用场景**：
- ✅ 产品三维分析（销售额、利润、销量）
- ✅ 客户价值分析
- ✅ 投资回报分析

**示例**：
```
数据源：产品分析
X 轴：销售额
Y 轴：利润率
气泡大小：订单数
结果：每个产品一个气泡，大小代表订单数
```

- ✅ 利润变化分析

- ✅ 成本构成变化

### 选择图表的原则

| **趋势分析** | 折线图、面积图、瀑布图 |
| **占比分析** | 环形饼图、漏斗图 |
| **分布分析** | 直方图、箱线图、小提琴图 |
| **关系分析** | 散点图、热力图 |
| **误差分析** | 误差棒图 |
| **多维分析** | 极坐标图（雷达图） |

### 最佳实践

1. **简洁明了** - 一图一主题
2. **数据标签** - 显示关键数值
3. **配色统一** - 使用企业配色
4. **标题清晰** - 说明图表内容
5. **适度装饰** - 避免过度设计
