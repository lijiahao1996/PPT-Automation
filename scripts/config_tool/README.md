## 📊 配置流程

### 步骤 1：统计规则配置

**操作**：
1. 打开"📋 统计规则配置"标签页
2. 填写表格名称（如：销售员业绩）
3. 选择统计类型（如：排名统计）
4. 填写分组字段（如：销售员）
5. 配置统计指标
6. 点击"添加统计规则"

**示例配置**：
```
表格名称：销售员业绩
统计类型：排名统计
分组字段：销售员
统计指标：[{"field": "销售额", "agg": "sum", "alias": "总销售额"}]
```

### 步骤 2：图表配置

**操作**：
1. 打开"📈 图表配置"标签页
2. 填写图表 Key（如：sales_by_person）
3. 填写图表标题
4. 选择图表类型
5. 填写数据源（必须与统计规则名称一致）
6. 配置字段
7. 点击"添加图表配置"

**示例配置**：
```
图表 Key：sales_by_person
图表标题：销售员业绩表现分析
图表类型：横向条形图
数据源：销售员业绩
X 轴字段：总销售额
Y 轴字段：销售员
```

### 步骤 3：数据预览

**操作**：
1. 运行 `Run.bat` 生成数据
2. 打开"📊 数据预览"标签页
3. 选择 Sheet 查看数据
4. 验证字段名称是否正确

### 步骤 4：导出配置

**操作**：
1. 打开"💾 导出配置"标签页
2. 点击"保存 stats_rules.json"
3. 点击"保存 placeholders.json"
4. 运行 `Run.bat` 生成 PPT

## 📋 最佳实践 Demo

### 完整配置示例

**场景**：新增"客户城市分析"图表

**1. 配置统计规则**（tabs 1）：
```
表格名称：客户城市分析
统计类型：排名统计
描述：客户属性与城市交叉分析
分组字段：
  客户属性
  城市
统计指标：
  [
    {"field": "销售额", "agg": "sum", "alias": "总销售额"},
    {"field": "订单数", "agg": "count", "alias": "订单数"}
  ]
```

**2. 配置图表**（tabs 2）：
```
图表 Key：customer_city
图表标题：客户属性与城市销售分析
图表类型：横向条形图
数据源：客户城市分析
X 轴字段：总销售额
Y 轴字段：城市
描述：客户属性与城市交叉分析
```

**3. 导出配置**（tabs 4）：
- 点击"保存 stats_rules.json"
- 点击"保存 placeholders.json"

**4. 配置 PPT 模板**：
- 打开 `templates/销售分析报告_标准模板.pptx`
- 找到要放置图表的页面
- 插入文本框，输入：`[CHART:customer_city]`
- 保存 PPT

**5. 运行生成**：
```bash
Run.bat
```

**完成！** 🎉

## 🎨 图表类型参考

| 图表类型 | 用途 | 需要字段 |
|---------|------|---------|
| bar_horizontal | 横向条形图 | x_field, y_field |
| bar_vertical | 纵向柱状图 | x_field, y_field |
| pie | 环形饼图 | category_field, value_field |
| column_clustered | 多列柱状图 | category_field, series |
| line | 折线图 | x_field, y_field |
| heatmap | 热力图 | index_field, columns |

## ❓ 常见问题

**Q: 图表不显示？**  
A: 检查数据源名称是否与统计规则一致

**Q: 字段找不到？**  
A: 在"数据预览"中查看实际生成的字段名

**Q: 如何修改配置？**  
A: 展开配置项，点击"编辑"按钮

**Q: 安装失败？**  
A: 检查 Python 是否安装，或尝试以管理员身份运行
