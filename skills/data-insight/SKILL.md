## ✍️ 洞察生成核心规则

### 1. 必须用数据说话
- ❌ 错误："销售额表现良好"
- ✅ 正确："Q2 销售额 21.3 万元，环比 Q1 增长 190%"

### 2. 必须有对比
- 同比/环比、目标 vs 实际、头部 vs 尾部、实际 vs 平均

### 3. 必须具体
- 精确到具体数字、具体人名、具体产品、具体城市

### 4. 必须有洞察
- 不仅描述"是什么"，还要指出"为什么"和"意味着什么"

### 5. 文案质量要求 ⭐⭐⭐⭐⭐

#### 字数要求（严格执行）
- **第 4 页（核心指标）**: 120-180 字
- **第 5-11 页（图表洞察）**: 每条 50-100 字，3 条合计 150-300 字
- **第 12 页（核心结论）**: 每段 60-100 字，4 段合计 240-400 字
- **第 13 页（落地策略）**: 每段 80-120 字，4 段合计 320-480 字

#### 内容要求
- 使用完整句子，不要简单罗列数据
- 包含**数据 + 对比 + 洞察**三层结构
- 避免"显著"、"明显"等模糊词汇，用具体倍数/百分比
- 每条洞察必须有**业务含义**或**行动建议**

#### 错误示例 ❌
```• 孙林以 39,601 元居销售员榜首
• 袁成洁与韩文分列 2、3 位
• TOP5 销售员合计贡献 62.9% 销售额```

#### 正确示例 ✅
```• 孙林以 39,601 元居销售冠军，其单笔最高订单 22,092 元占个人业绩 55.8%，依赖大单明显，业绩结构存在风险
• 袁成洁与韩文业绩相近（36,378 元 vs 36,113 元），但袁成洁客单价更高（3,307 元 vs 3,283 元），转化效率略优
• TOP5 销售员合计贡献 179,590 元（占比 62.9%），头部效应显著，但孙林大单依赖可能影响团队稳定性```

### 各页面具体要求

#### 第 4 页：核心经营指标总览
**格式**：单条完整洞察，100-150 字
**内容**：总销售额、订单数、客单价、极值、整体趋势判断

**示例**:
```
总销售额 285,696 元中，Q2 贡献 212,539 元（占比 74.4%），环比 Q1 增长 190.5%；月度销售在 6 月达峰值 103,640 元，占半年总额 36.3%，单月增速较 5 月提升 75.3%；平均客单价 2,856.96 元，但最低单额为 0 元，提示存在无效订单或系统录入异常，需核查订单质量。
```

---

#### 第 5 页：销售员业绩表现分析
**格式**：**3 条列表式洞察**，用 `•` 符号分隔，每条独立一行
**内容要求**:
- 第 1 条：TOP3 销售员排名及具体销售额
- 第 2 条：头部贡献占比（如 TOP3/总销售额）
- 第 3 条：客单价差异分析（头部 vs 平均）

**示例** (每条 50-100 字):
```
• 孙林以 39,601 元居销售冠军，其单笔最高订单 22,092 元占个人业绩 55.8%，依赖大单明显；袁成洁与韩文分列 2、3 位（36,378 元 vs 36,113 元），差距仅 265 元，竞争激烈
• TOP3 销售员合计贡献 112,092 元（占比 39.2%），头部效应显著；但孙林大单依赖可能影响团队稳定性，需关注业绩可持续性
• TOP5 销售员客单价集中在 3,060-3,307 元区间，极差仅 247 元，说明头部销售员能力接近；王伟客单价最低（3,060 元），需提升高价值客户转化能力
```

---

#### 第 6 页：产品销售结构与占比分析
**格式**：**3 条列表式洞察**，用 `•` 符号分隔
**内容要求**:
- 第 1 条：爆品识别（TOP1 产品及占比）
- 第 2 条：核心品类集中度（TOP3/4 合计占比）
- 第 3 条：长尾品类问题及建议

**示例** (每条 50-100 字):
```
• 牛肉干以 97,718 元、34.2% 占比绝对主导，超第二名柳橙汁（56,852 元）72%，单品拉动效应显著；但过度依赖单一品类存在波动风险，需建立抗风险产品矩阵
• 柳橙汁（19.9%）与巧克力（15.0%）合计占 34.9%，接近牛肉干单品类份额；4 大核心品类（牛肉干 + 柳橙汁 + 巧克力 + 苹果汁）贡献 84.7% 销售额，结构集中度较高
• 可乐仅占 5.6%（15,999 元）在 6 类产品中垫底，且无城市/销售员聚焦数据支撑；牛奶占比 11.4%，二者合计 17% 属长尾品类，建议通过组合捆绑提升连带率
```

---

#### 第 7 页：城市区域销售业绩分析
**格式**：**3 条列表式洞察**，用 `•` 符号分隔
**内容要求**:
- 第 1 条：TOP3 城市及合计占比
- 第 2 条：头部尾部差距（倍数关系）
- 第 3 条：潜力市场识别

**示例** (每条 50-100 字):
```
• 莆田以 39,601 元与销售冠军孙林业绩完全一致，高度疑似其主攻城市，人城强绑定；龙岩（36,378 元）与福州（36,113 元）仅差 265 元，但对应销售员不同，呈现'人 - 城 - 绩'精准映射
• TOP3 城市（莆田 + 龙岩 + 福州）合计贡献 112,092 元（占比 39.2%），头部效应明显；头部城市与尾部城市业绩差距达 2.1 倍，区域发展不均衡问题突出
• 三明、宁德等尾部城市市场渗透率不足，月均销售额低于 10,000 元；建议针对尾部城市制定专项拓展计划，复制莆田 - 孙林成功模式，提升区域覆盖率
```

#### 第 8 页：新老客销售贡献对比分析
**格式**：**3 条列表式洞察**，用 `•` 符号分隔
**内容要求**:
- 第 1 条：老客贡献占比及金额
- 第 2 条：客单价对比（老客 vs 新客倍数）
- 第 3 条：新客转化问题及建议

**示例** (每条 50-100 字):
```
• 老客贡献 235,424 元（82.4%），是新客（50,272 元）的 4.7 倍，复购为基本盘；但过度依赖老客存在风险，一旦老客流失将导致业绩大幅波动，需建立客户多元化结构
• 老客客单价 3,859.41 元，是新客（1,289 元）的 3 倍，说明老客信任度高、购买力强；新客首单金额偏低，可能受试用装/小规格产品影响，需优化首单产品组合策略
• 新客占比 17.6% 低于行业健康值（通常≥30%），6 个月累计仅 50,272 元，月均 8,379 元；拉新能力严重不足，获客渠道或激励机制可能失效，需紧急优化
```

#### 第 9 页：销售时间趋势分析
**格式**：**3 条列表式洞察**，用 `•` 符号分隔
**内容要求**:
- 第 1 条：月度趋势（峰值月份及环比）
- 第 2 条：季度对比（Q1 vs Q2 占比）
- 第 3 条：星期分布（峰值/谷值及建议）

**示例**:
```
• 6 月销售额 103,640 元为半年峰值，占 Q2 总额 48.8%
• Q2 贡献 74.4% 销售额，环比 Q1 增长 190.5%，季节性特征明显
• 周日销售额最低，建议针对周末推出专属促销活动
```

#### 第 10 页：销售员 - 产品热力图分析
**格式**：**3 条列表式洞察**，用 `•` 符号分隔
**内容要求**:
- 第 1 条：TOP 销售员的擅长品类及占比
- 第 2 条：特定销售员在特定品类的突出表现
- 第 3 条：尾部销售员的产品推广能力问题

**示例**:
```
• 孙林、袁成洁、韩文三人合计销售 112,092 元，占总额 39.2%
• 张珊、王伟分列销售第 4、5，二人所在城市亦为城市第 4、5
• 异常订单中孙阳未进销售 TOP5，却贡献单笔高值，提示非头部人员存在潜力未释放
```

#### 第 11 页：异常订单专项排查
**格式**：**3 条列表式洞察**，用 `•` 符号分隔
**内容要求**:
- 第 1 条：异常订单数量、金额、集中度
- 第 2 条：最大异常订单详情及占比
- 第 3 条：风险提示（0 元订单、数据质量等）

**示例**:
```
• 六个月中仅 6 月突破 10 万元，其余均低于 6 万元，业绩稳定性严重不足
• 客单价均值 2,856.96 元，但 TOP5 销售员客单价集中在 3,060–3,307 元区间
• 最低单额 0 元出现于某销售员或某城市，指向特定环节操作漏洞或考核误导
```

#### 第 12 页：全量分析核心洞察结论 ⭐
**格式**：**4 条结构化洞察**，用 `\n\n` 分隔，每条有明确小标题
**内容维度**（必须覆盖）:

1. **业绩结构**：分析销售额的构成特征
2. **增长亮点**：指出表现优异、值得复制的亮点
3. **核心短板**：指出明显的问题和短板
4. **业务风险**：指出潜在风险

**示例格式**:
```
业绩结构：老客贡献 82.4%（235,424 元），新客仅 17.6%（50,272 元），收入结构高度依赖存量，抗风险能力弱

增长亮点：Q2 销售额 212,539 元，环比 Q1 增长 190.5%，其中 6 月单月破 10 万（103,640 元），占半年总额 36.3%，呈现爆发式增长

核心短板：牛肉干一家独大（34.2%），第二名柳橙汁仅 19.9%，CR2=54.1%，品类集中度过高，抗单品波动能力差

业务风险：存在 3 笔异常大单（孙林 22,092 元、袁成洁 13,650 元、孙阳 12,364 元），合计 48,106 元占总额 16.8%，且最低单额为 0 元，订单质量管控失效
```

#### 第 13 页：可落地销售优化策略建议 ⭐
**格式**：**4 条结构化策略**，用 `\n\n` 分隔，每条有明确策略名称
**内容维度**（必须覆盖）:

1. **客户运营策略**：老客维系 + 新客转化的具体做法
2. **产品组合策略**：爆品带动 + 滞销品促销的具体做法
3. **团队管理策略**：经验分享 + 培训激励的具体做法
4. **营销节奏策略**：时间维度（周末/月度节点）的具体做法

**示例格式**:
```
客户运营策略：针对老客启动高净值分层计划，将客单价≥3,300 元客户纳入 VIP 服务池，配置专属顾问并试点年度订阅制

产品组合策略：设立健康双擎攻坚小组，以柳橙汁 + 苹果汁为组合包（当前合计 33.8%），在福州、厦门试点果汁 + 牛肉干满赠策略，对冲牛肉干单一依赖

团队管理策略：推行新锐孵化制，从孙阳等非 TOP 但有大单潜力人员中选拔 3 人，匹配莆田/龙岩成熟团队带教，并设置新客首单奖（500 元/单）

营销节奏策略：构建 3+1 淡旺季模型，Q1 每月设定 30% 基线目标 +1 次主题快闪（如 2 月年货节、3 月开学期），6 月成果反哺 Q3 资源储备，避免业绩断崖
```

## 🔧 技术实现要求

### JSON 输出格式

```json
[
  "总销售额 285,696 元中，Q2 贡献...（100-150 字）",
  "• 孙林以 39,601 元总销售额位列业绩 TOP1...\n• TOP3 销售员贡献了全团队 39% 的销售额...\n• 头部销售员客单价显著高于团队平均水平...",
  "• 牛肉干为核心爆品，销售额 97,753 元...\n• 柳橙汁、巧克力、苹果汁位列 2-4 位...\n• 牛奶、可乐占比不足 12%...",
  ...
]
```

### 关键要求

1. **必须是 10 条字符串**，不能多不能少
2. **不能嵌套数组**，所有洞察都必须是字符串
3. 列表式洞察用 `\n` 连接 3 条，如：`"• 洞察 1\n• 洞察 2\n• 洞察 3"`
4. 结构化洞察用 `\n\n` 分隔 4 个段落
5. 直接输出 JSON 数组，不要任何其他文字

### 注意事项

1. 所有数字使用阿拉伯数字，千位加分隔符（如 39,601）
2. 百分比保留 1 位小数（如 34.2%）
3. 金额单位统一用"元"或"万元"
4. 人名、产品名、城市名必须与数据一致
5. 列表式洞察用 `•` 符号，每条换行
6. 结构化洞察用小标题 + 冒号格式
