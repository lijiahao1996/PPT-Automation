## 📊 使用配置工具创建散点图

### 步骤 1：打开配置工具

访问：http://localhost:8501/

### 步骤 2：添加统计规则

切换到 **"📋 统计规则配置"** 标签页：

**填写内容**:
```
统计表格名称：客户年龄分布
统计类型：📊 分布分析 - 星期分布
启用：✅
描述：客户年龄与销售额分布分析

分组字段（每行一个）：
年龄段
客户类型

统计指标配置（JSON 格式）：
[
  {"field": "销售额", "agg": "sum", "alias": "总销售额"},
  {"field": "销售额", "agg": "count", "alias": "订单数"},
  {"field": "销售额", "agg": "mean", "alias": "客单价"}
]
```

点击 **"➕ 添加统计规则"** → ✅ **自动保存**（无需手动点击保存按钮）

---

### 步骤 3：添加散点图配置

切换到 **"📈 图表配置"** 标签页：

**填写内容**:
```
图表 Key：age_scatter
图表标题：客户年龄与销售额分布分析
图表类型：⚡ 散点图
数据源：客户年龄分布（下拉选择）

X 轴字段：年龄段
Y 轴字段：总销售额
描述：分析不同年龄段客户的销售贡献和订单分布
```

> 💡 **提示**：散点图现在支持直接输入 X 轴和 Y 轴字段，不需要 JSON 格式了！

点击 **"➕ 添加图表配置"** → ✅ **自动保存**（无需手动点击保存按钮）

---

### 步骤 4：添加洞察配置

切换到 **"💡 洞察配置"** 标签页：

找到 `CHART:age_scatter`，配置如下：

**分析维度**（多选）：
- ✅ 分布分析
- ✅ 对比分析
- ✅ 相关性分析

**关键指标**：
```
总销售额
客单价
订单数
```

**对比基准**：平均值

**洞察风格**：数据驱动

**字数要求**：150 字

**自定义提示词**：
```
重点分析哪个年龄段客户贡献最高，年轻客户（25-35 岁）与中年客户（36-50 岁）的消费差异，以及客户类型（新客/老客）在各年龄段的分布特征。指出潜在的年龄分层运营机会。
```

点击 **"💾 保存洞察配置"** → ✅ **自动保存**
