## 6. 修改 PPT 模板

### 6.1 打开模板文件

位置：`templates/销售分析报告_标准模板.pptx`

使用 PowerPoint 打开。

### 6.2 插入占位符

#### 6.2.1 文本占位符

在需要显示文本的位置插入文本框，输入：

```
[TEXT:report_title]
```

**说明**：会自动替换为"销售数据分析报告"

#### 6.2.2 图表占位符

在需要显示图表的位置插入文本框，输入：

```
[CHART:sales_by_person]
```

**说明**：会自动替换为销售员业绩条形图

#### 6.2.3 AI 洞察占位符

在需要显示 AI 分析的位置插入文本框，输入：

```
{{INSIGHT:sales_by_person}}
```

**说明**：会自动替换为销售员业绩的 AI 分析

#### 6.2.4 结论和策略占位符

在结论页插入文本框，输入：

```
{{INSIGHT:conclusion}}
```

在策略页插入文本框，输入：

```
{{INSIGHT:strategy}}
```

### 6.3 保存模板

修改完成后，保存 PPT 模板文件。

## 💡 小贴士

1. **数据准备**：确保 Excel 数据包含必要的列（如：订单时间、销售员、产品、销售额等）
2. **统计规则**：根据数据字段灵活配置，不必拘泥于示例
3. **图表配置**：数据源必须与统计规则的名称一致
4. **AI 洞察**：自定义提示词可以让分析更有针对性
5. **PPT 模板**：占位符格式要正确，注意区分 `[CHART:]` 和 `{{INSIGHT:}}`

## ❓ 常见问题

**Q: 上传文件后没有反应？**  
A: 检查文件格式是否为 .xlsx 或 .xls

**Q: 统计汇总生成失败？**  
A: 检查 Excel 数据是否包含必要的列

**Q: 图表不显示？**  
A: 检查数据源名称是否与统计规则一致

**Q: AI 洞察没有生成？**  
A: 检查 API Key 是否正确配置

**Q: PPT 占位符没有替换？**  
A: 检查占位符格式是否正确（注意中括号和大括号）
